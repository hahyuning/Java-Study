## 1. 이진 트리 종류

### 이진 트리, 이진 탐색 트리
- 이진 트리: 각 노드가 최대 두 개의 자식을 갖는 트리
- 이진 탐색 트리: 모든 노드가 특정 순서를 따르는 속성이 있는 이진 트리
  - 모든 왼쪽 자식 노드 <= 부모 노드 <= 모든 오른쪽 자식 노드

### 균형 트리
- 모든 노드에 대해서 왼쪽 부분 트리의 높이와 오른쪽 부분 트리의 높이의 차이가 최대 1
- O(logN) 시간에 insert와 find를 할 수 있을 정도
- 레드-블랙 트리, AVL 트리

### 완전 이진 트리 (complete binary tree)
- 트리의 모든 높이에서 노드가 꽉 차있는 이진 트리
- 마지막 단계는 꽉 차있지 않아도 되지만 노드가 왼쪽에서 오른쪽으로 채워져야 한다.

### 전 이진 트리 (full binary tree)
- 모든 노드의 자식이 없어가 정확히 두 개 있는 경우
- 자식이 하나만 있는 노드가 존재해서는 안 된다.

### 포화 이진 트리 (perfect binary tree)
- 전 이진 트리이면서 완전 이진 트리인 경우
- 모든 말단 노드는 같은 높이에 있어야 하며, 마지막 단계에서 노드의 개수가 최대가 되어야 한다.
- 노드의 개수가 정확히 2^(k-1) 개여야 한다.

## 2. 이진 트리 순회
### 중위 순회 (in-order)
- 왼쪽 가지, 현재 노드, 오른쪽 가지 순서로 노드를 방문하고 출력하는 방법
- 이진 탐색 트리를 중위 순회하면 오름차순으로 방문

### 전위 순회 (pre-order)
- 자식 노드보다 현재 노드를 먼저 방문하는 방법
- 전위 순회에서 가장 먼저 방문하는 노드는 언제나 루트

### 후위 순회 (post-order)
- 모든 자식 노드들을 먼저 방문한 뒤 마지막에 현재 노드를 방문하는 방법
- 후위 순회에서 가장 마지막에 방문하는 노드는 언제나 루트
